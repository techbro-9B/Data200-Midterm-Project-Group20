# Data200-Data-Analysis-Projects-Group20

by: Matie G, Matie V, Kim K, Sidon M & Lloyd N. 


## The Midterm Project.
This project centers around answering one research question, how well can house prices be predicted using property-specific features and macroeconomic data? Which features micro (property details) or macro (House Price Index) trends contribute most to price variation?
The housing data is taken from the Ames, Iowa, USA and the economic index data is sourced from the federal reserves of the United States of America.
